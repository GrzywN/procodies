## Run Locally

Clone the project

```bash
  git clone https://github.com/GrzywN/procodies
```

Go to the project directory

```bash
  cd procodies
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** Astro, Vue, TailwindCSS
